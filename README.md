<h1>Versatile Responsive Menu</h1>

<h2>Introduction</h2>

Welcome to the Versatile Responsive Menu! This is a highly flexible, robust, and developer-friendly 
navigation solution designed for modern web environments. Whether you are building a lightning-fast 
static HTML website or developing a custom, dynamic WordPress theme, this menu provides the perfect 
structural foundation for a seamless user experience across all devices.

<h2>Description</h2>

What makes the Versatile Responsive Menu unique is its lightweight footprint combined with unprecedented 
flexibility. Instead of relying on bloated CSS frameworks, this script utilizes a single, smart HTML 
attribute: data-menu-style. By simply changing the value of this attribute, the menu transforms on the 
fly between completely different layout structures.

Furthermore, the system is specifically built with content management systems in mind. It offers native support 
for the WordPress <strong>wp_nav_menu()</strong> function and integrates seamlessly with Advanced Custom Fields (ACF).
With the included PHP script, you empower website administrators to change the menu type directly from 
the WordPress backend, without having to touch a single line of code.

<h2>Features & Capabilities</h2>

Four Unique Menu Types: Switch instantly between the following styles via the data-menu-style attribute:

<ul>
<li>horizontal: The classic horizontal dropdown menu (most commonly used for standard websites).</li>

<li>vertical: A vertical flyout menu (ideal for dashboards and complex web apps).</li>

<li>accordion: A vertical accordion menu (perfect for mobile screens and sidebars).</li>

<li>megamenu: A wide, multi-column megamenu (essential for e-commerce and large-scale platforms).</li>
</ul>

<p><strong>100% Responsive:</strong> The menu scales smoothly from a wide desktop layout to an intuitive hamburger menu (.menu-toggle-btn) for mobile users.</p>

<p><strong>Seamless WordPress Integration:</strong>  Ready to use in any custom WordPress theme thanks to the optimized PHP templates.</p>

<p><strong>Dynamic Management (ACF Ready):</strong>  Link the menu to a 'Select' field in ACF, allowing administrators to visually manage the layout directly from the backend.</p>

<p><strong>Multi-level Support:</strong>  Developed and tested for deep tree structures up to three levels (one main level and two sub-levels), compliant with current UI/UX standards.</p>

<p><strong>Absolute Developer Freedom:</strong>  The HTML output contains no unnecessary classes or inline styles. This grants you complete control over the CSS styling and makes it incredibly easy to add custom JavaScript animations or ARIA accessibility attributes.</p>
